# Flask-Blog

Project still in early stages.

## Install
    git clone https://github.com/mharrys/flask-blog.git
    cd flask-blog
    virtualenv venv
    source venv/bin/activate
    pip install -r requirements.txt
    python manage.py init

## Test
    python tests.py

## Run
    python manage.py runserver
